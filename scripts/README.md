# Scripts tree

This repository now exposes organized platform launcher folders.

## Preferred platform folders

- `scripts/linux/`
- `scripts/windows/`
- `scripts/macos/`

## Runtime rule

These launchers stay thin.

The operational truth remains:

```bash
python run.py ...
```

## Compatibility note

Older root-level launchers still remain under `scripts/`.

They are kept for compatibility while the organized platform folders become the preferred public reference path.
